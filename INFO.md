## 🔥 الخطوات الأساسية:

```bash
# 1. تثبيت Socketio
cd Socketio && npm install

# 2. تثبيت Client
cd ../Client && npm install

# 3. تشغيل السيرفر (Terminal 1)
cd Socketio && npm start

# 4. تشغيل التطبيق (Terminal 2)
cd Client && npm run dev

# 5. افتح المتصفح
http://localhost:5173
```
