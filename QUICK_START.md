# 🎯 دليل البدء السريع - ELFAR LIVE CHAT

**هذا الدليل سهل جداً! اتبع الخطوات بالترتيب** 👇

---

## ✅ قبل ما تبدأ - تأكد من:

- [ ] عندك Node.js مثبت (اطلع على عنوان: https://nodejs.org)
- [ ] عندك متصفح إنترنت (Chrome أو Firefox)
- [ ] الـ Command Line/Terminal (أسود اللون بتاع الحاسب)

---

## 🚀 الخطوات:

### 📍 الخطوة 1: فتح مجلد المشروع

```bash
cd d:\2026\Projects\react-socketio
```

---

### 📍 الخطوة 2: تثبيت المتطلبات للسيرفر

```bash
cd Socketio
npm install
```

**ما يحدث:** الحاسب يحمل البرامج المساعدة للسيرفر
**الوقت:** دقيقة أو دقيتين

---

### 📍 الخطوة 3: تثبيت المتطلبات للواجهة

```bash
cd ../Client
npm install
```

**ما يحدث:** الحاسب يحمل البرامج المساعدة للواجهة
**الوقت:** دقيقة أو دقيتين

---

### 📍 الخطوة 4: تشغيل السيرفر (Terminal 1)

**افتح Terminal/Command Line جديد وادخل:**

```bash
cd d:\2026\Projects\react-socketio\Socketio
npm start
```

**ما يحدث:** السيرفر يشتغل على المنفذ 3000

**علامات نجاح:**

```
✅ Server running on port 3000
✅ Ready for connections
```

**بلاش هذا الـ Terminal! دعه يشتغل** 🔄

---

### 📍 الخطوة 5: تشغيل التطبيق (Terminal 2)

**افتح Terminal/Command Line **جديد** وادخل:**

```bash
cd d:\2026\Projects\react-socketio\Client
npm run dev
```

**ما يحدث:** التطبيق يشتغل على عنوان `http://localhost:5173`

**علامات نجاح:**

```
✅ VITE v4.x.x ready in xxx ms
✅ ➜  Local:   http://localhost:5173
```

**انسخ العنوان وأفتحه في المتصفح** 🌐

---

### 📍 الخطوة 6: الاستخدام

1. 📱 المتصفح سيفتح تلقائياً
2. ✏️ اكتب اسمك في الحقل (مثل: أحمد)
3. 🎯 اضغط "ادخل الدردشة"
4. 💬 ابدأ الكتابة!

**تحتاج تجرب مع شخص آخر؟**

افتح عنوان جديد: `http://localhost:5173`
اسم مختلف واكتب الرسائل بينهم 😊

## 🛑 إذا حصلت مشكلة:

### ❌ "Port 3000 is already in use"

```bash
# السيرفر يعمل بالفعل
# اغلق التطبيق ثم حاول مجدداً
```

### ❌ "npm not found"

```bash
# Node.js ما بتثبتش صحيح
# حمل من: https://nodejs.org
```

### ❌ "Cannot find module"

```bash
# اشتغل npm install مرة ثانية
cd Socketio
npm install

cd ../Client
npm install
```

### ❌ "لا يمكن الاتصال بالسيرفر"

```bash
# تأكد إن السيرفر يشتغل
# شف الـ Terminal الأول (بتاع Socketio)
# يجب يكون فيه رسالة "Server running on port 3000"
```

---

## 💡 نصائح مهمة:

1. **لا تغلق أي Terminal** - احتاج كلاهما يشتغل
2. **بلاش الـ Refresh** - اضغط F5 لتحديث الصفحة
3. **الإنترنت مهم** - التطبيق ما يشتغل بدون إنترنت حتى محلياً
4. **السيرفر أولاً** - ابدأ بالسيرفر قبل التطبيق
