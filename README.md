## 📋 نظرة عامة على المشروع

**ELFAR LIVE CHAT** هو مشروع متكامل يوفر منصة دردشة حية وفورية للمستخدمين. يتكون المشروع من جزأين رئيسيين:

### 🎯 الخدمات الرئيسية:

| الخدمة                 | الوصف                              |
| ---------------------- | ---------------------------------- |
| 💬 **دردشة فورية**     | إرسال واستقبال الرسائل بدون تأخير  |
| 👥 **قائمة الأعضاء**   | رؤية جميع المستخدمين المتصلين الآن |
| ⌨️ **تنبيهات الكتابة** | معرفة من يكتب الآن في الغرفة       |
| 🔗 **حالة الاتصال**    | مؤشر واضح لحالة الاتصال بالسيرفر   |
| 📊 **العداد**          | عرض عدد المستخدمين المتصلين        |
| 🎨 **واجهة جميلة**     | تصميم عصري وسهل الاستخدام          |

## 🚀 البدء السريع

### المتطلبات:

- Node.js 16+
- npm 7+

### الخطوة 1: التثبيت

```bash
# تثبيت السيرفر الخلفي
cd Socketio
npm install

# تثبيت الواجهة الأمامية
cd ../Client
npm install
```

### الخطوة 2: إعداد البيئة

```bash
# في مجلد Client
cp .env.example .env

# تأكد من أن VITE_SOCKET_SERVER_URL يشير إلى السيرفر المحلي
# VITE_SOCKET_SERVER_URL=http://localhost:3000
```

### الخطوة 3: تشغيل المشروع

**تشغيل السيرفر:**

```bash
cd Socketio
npm start
# سيشتغل على: http://localhost:3000
```

**تشغيل الواجهة الأمامية (في terminal جديد):**

```bash
cd Client
npm run dev
# سيشتغل على: http://localhost:5173
```

### الخطوة 4: الاستخدام

1. افتح المتصفح على `http://localhost:5173`
2. أدخل اسمك
3. اضغط "ادخل الدردشة"
4. ابدأ الدردشة! 💬

اختر أحد هذه الخيارات:

## 🐛 استكشاف الأخطاء

### المشكلة: لا يمكن الاتصال بالسيرفر

**الحل:**

- تأكد من أن السيرفر يعمل على المنفذ 3000
- تحقق من `VITE_SOCKET_SERVER_URL` في ملف `.env`

### المشكلة: الرسائل لا تظهر

**الحل:**

- أعد تحميل الصفحة
- تأكد من أن الاتصال نشط (الشاشة الخضراء)

### المشكلة: النمط CSS غير صحيح

**الحل:**

```bash
cd Client
npm install
npm run dev
```
